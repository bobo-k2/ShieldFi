FAQ

Frequently asked

What is ShieldFi?

ShieldFi is a free AI-powered Solana wallet security scanner. Paste any wallet address to detect suspicious tokens, analyze contract risks, check for impersonation, and review transaction history — no wallet connection needed.

Is ShieldFi free to use?

Yes. Basic scanning is completely free forever, no signup required. Premium tiers add real-time monitoring, Telegram alerts, and advanced AI analysis starting at $7/month. See our pricing page for details.

Do I need to connect my wallet?

No. ShieldFi never requires a wallet connection or transaction signing. Just paste a public Solana address and we analyze it using on-chain data. Your private keys always stay private.

What threats does ShieldFi detect?

Fake token impersonation, suspicious authorities (mint/freeze), risky approvals, known scam tokens, unusual patterns, and smart contract vulnerabilities — all using AI-powered analysis.

Which wallets are supported?

Any Solana wallet — Phantom, Solflare, Backpack, Ledger, or any other. ShieldFi reads public on-chain data, so it works with every Solana address.
